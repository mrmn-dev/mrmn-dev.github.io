IBM MQ

IBM MQ is an Enterprise Integration product that supports application/platform integration via either the IBM MQ messaging API or JMS. There is a developer edition of the product that is available for download and installation. IBM MQ has evolved and changed over the years to support a broad range of application types.

Using the IBM MQ installation documentation and rundeck community edition the steps required to install IBM MQ on Fedora were created. The rundeck directory was set up to see how one might use rundeck to support deploying and operating MQ. At this point have I have just set up some simple jobs and am exploring how rundeck may be used to manage environments at scale. More thoughts on this will be forthcoming. Until then, you can use the following link and import the jobs in the rundeck directoy into your own environment. Link to miscMQ github repository.
