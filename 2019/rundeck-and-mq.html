Sat 23 February 2019

IBM MQ

IBM MQ is an Enterprise Integration product that supports application/platform integration via either the IBM MQ messaging API or JMS. There is a developer edition of the product that is available for download and installation. IBM MQ has evolved and changed over the years to support a broad range of application types.

Using the IBM MQ installation documentation and rundeck community edition, the steps required to install IBM MQ on Fedora were created. The rundeck directory was set up to see how one might use rundeck to support deploying and operating MQ. At this point have I have just set up some simple jobs to explore how rundeck may be used to manage environments at scale. You can use the link below and import the jobs in the rundeck directoy into your own environment.
